# Swiss Tourism Data Analysis

A data analysis project using **Python** and **PostgreSQL**.  
The goal is to explore open-source data about Swiss tourism, clean and transform it, and create visualizations with **PostGIS** and **Excel/Power BI**.

---

## Project Goals
- Practice **data cleaning** and **analysis** workflows.  
- Use **PostgreSQL + PostGIS** for data management and geospatial analysis.  
- Apply **Python** for ETL (Extract, Transform, Load) tasks.  
- Build **visualizations** in Excel or Power BI.  

---

## Project Structure
- `data/` Original datasets (`.csv`)  
- `etl_python_script.py` Python ETL script  
- `sql_queries/`SQL queries (saved as `.csv`)
